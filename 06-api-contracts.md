# 06 - API Contracts

## 1. Authentication Endpoints
All protected endpoints require header: `Authorization: Bearer <supabase_jwt>`.

### `GET /auth/profile/{user_id}`
- **Response:** `200 OK`
- **Body:**
  ```json
  {
    "id": "uuid",
    "full_name": "string",
    "target_accent": "Indian English | British English",
    "current_level": "beginner | intermediate | advanced",
    "daily_streak": 5,
    "total_sessions": 42
  }
  ```

## 2. Session Management

### `POST /sessions/start`
- **Request Body:** `{"user_id": "uuid", "target_accent": "string", "level": "string"}`
- **Response:**
  ```json
  {
    "session_id": "uuid",
    "greeting": "Hello! How can I help you today?",
    "greeting_audio": "base64_wav_string",
    "first_exercise": "Describe your morning routine in 3 sentences."
  }
  ```

### `POST /sessions/turn`
- **Request Body:**
  ```json
  {
    "session_id": "uuid",
    "user_id": "uuid",
    "audio_base64": "string",
    "turn_number": 1
  }
  ```
- **Response:**
  ```json
  {
    "transcript": "string",
    "grammar_score": 85.5,
    "vocabulary_score": 70.0,
    "confidence_score": 65.2,
    "fluency_score": 75.0,
    "wpm": 135,
    "filler_count": 2,
    "ai_response": {
      "response_text": "string",
      "praise": "string",
      "tip": "string",
      "exercise": "string"
    },
    "ai_audio_base64": "string"
  }
  ```

### `POST /sessions/end`
- **Request Body:** `{"session_id": "uuid", "user_id": "uuid"}`
- **Response:** Returns aggregated session scores and detailed feedback summary.

## 3. Progress Tracking

### `GET /progress/{user_id}?days=7`
- **Response:** Array of daily score snapshots.

### `GET /progress/history/{user_id}?limit=10`
- **Response:** List of summaries for the last N sessions.

## 4. Interview (Phase 2)
### `POST /interview/setup`
- **Request Body:** Company name, role, panel size, rounds config.
- **Response:** Panel configuration generated by AI.
